# githubAPI
simple REST service to obtain data from github API 

Endpoints descriptions for user
| HTTP Method | URI |Description |
| --- | --- | --- |
| GET | `/users/{login}` | Get user with login |


## Technologies
* Java 17
* Spring Boot 
* Postgresql
* Gradle
* Spock


## Run Application
```
$ cd ../githubAPI
$ ./gradlew bootRun
```
